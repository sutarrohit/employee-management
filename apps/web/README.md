# Web — Employee Management Dashboard

Next.js 16 dashboard for managing employees and viewing workforce insights.

## Features

- Employee CRUD (create, read, update, delete)
- Dashboard with key metrics (total employees, payroll, salary distribution)
- Salary distribution visualization
- Top earners table
- Payroll breakdown by department
- Country-wise salary insights
- Employee filtering and searchable data table
- Dark mode support
- Drag-and-drop sortable tables
- Responsive sidebar layout

## Tech Stack

- **Next.js 16** (React 19)
- **Tailwind CSS 4** with shadcn/ui components
- **TanStack React Query** for server state management
- **TanStack Table** for data tables
- **Recharts** for charts and visualizations
- **React Hook Form + Zod** for form validation
- **dnd-kit** for drag and drop
- **next-themes** for dark/light mode

## Getting Started

```sh
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
